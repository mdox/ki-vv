# Kód Informatika - Valutaváltó

## Stack

-   React
-   TailwindCSS
-   Flowbite
-   useQuery
-   zustand
-   Express
-   Prisma
-   SQLite

## Usage

### Backend

```sh
$ cd ki_vv-backend

$ npm run dev

$ npm run build
$ npm run serve
```

### Frontend

```sh
$ cd ki_vv-frontend

$ npm run dev

$ npm run build
$ npm run preview
```
